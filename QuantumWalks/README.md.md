# QuantumWalks - Discrete- and Continuous- time Quantum Walks 

A specialized package for Discrete-Time (DTQW) and Continuous-Time (CTQW) Quantum Walk simulations in 1D and discrete billiard geometries.

## 🚀 Loading the Package 

To load the package in a Mathematica session, use the `Get` command pointing to the `init.m` file inside the Kernel: 

```mathematica
Get["path/to/libs/QuantumWalks/Kernel/init.m"]
```

## ❓Functions details and usage examples

See the notebooks in `UsageExamples/`.

## 🛠️ Dependencies

This package requires the **ForScience** library for usage message formatting.

- The package attempts to install it automatically if not found (`PacletInstall["ForScience"]`).
- Repository: [MMA-ForScience on GitHub](https://github.com/MMA-ForScience/ForScience).
